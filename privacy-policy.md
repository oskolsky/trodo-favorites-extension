# Privacy Policy for Trodo Favorites Extension

**Last updated:** 2026-04-02

## Overview

Trodo Favorites Extension is a browser extension that allows users to save and manage a list of favorite products from the Trodo website.

## Data Collection

This extension does **not collect, store, or transmit any personal data** to external servers.

All data is stored locally in the user's browser using Chrome's built-in storage (`chrome.storage.local`).

## Data Stored Locally

The extension stores the following information locally:

- Product ID
- Product name
- Product image URL
- Product price
- Product URL
- Currency
- Discount information (if available)

This data is used solely to provide the functionality of the extension.

## Data Sharing

We do not share, sell, or transmit any user data to third parties.

## Permissions Used

The extension may request the following permissions:

- **storage** — to store favorite products locally
- **activeTab** — to access the current Trodo page
- **host permissions (trodo.com)** — to read product information from Trodo pages

These permissions are used only for the core functionality of the extension.

## Third-Party Services

This extension does not use any third-party analytics, tracking, or external APIs.

## Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected in this document.
